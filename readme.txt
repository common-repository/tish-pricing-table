== Description ==

Tish Pricing Table is an intuitive WordPress plugin tailored for webmasters and businesses looking to showcase their services or products in a structured pricing format. Whether you're a startup introducing various subscription tiers or an established enterprise highlighting premium offerings, our plugin ensures a seamless creation and customization process. With features like responsive design, detailed tooltips, custom plan images, and advanced color customization, Tish Pricing Table not only enhances the visual appeal but also improves user engagement. Elevate your website's professionalism and user experience by integrating our state-of-the-art pricing table plugin.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the `tish-pricing-table` folder to the to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Navigate to Admin Panel -> 'Tish Pricing Table' -> fill pricing fields
4. insert the shortcode [tish_pricing_table] to display the pricing table in any page or post
5. Enjoy!
